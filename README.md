# CodeAlpha_Tasks

This repository contains my CodeAlpha internship tasks. Each project demonstrates different aspects of full‑stack development.

---

## 🛒 TASK 1: Simple E-commerce Store
**Goal:** Build a basic e-commerce site with product listings.

**Tech Stack:**
- Frontend: HTML, CSS, JavaScript
- Backend: Django (Python) or Express.js (Node.js)
- Database: For storing products, users, and orders

**Features:**
- Product listings
- Product details page
- Shopping cart
- Order processing
- User registration/login
- Database integration for products, users, and orders

---

## 🌐 TASK 2: Social Media Platform
**Goal:** Create a mini social media app.

**Tech Stack:**
- Frontend: HTML, CSS, JavaScript
- Backend: Django or Express.js
- Database: For users, posts, comments, followers

**Features:**
- User profiles
- Posts & comments
- Like system
- Follow/unfollow system
- Notifications (stored + real-time via Socket.IO)
